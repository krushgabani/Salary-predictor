# 💼 Salary Predictor

An ML-powered web app that predicts job salaries based on 
role, experience, location, and skills.

## 🚀 Live Demo
👉 https://krush-salary-predictor.streamlit.app

## 🔍 Model Details
- Algorithm: Random Forest Regressor
- Dataset: 250,000 job records
- Accuracy: R² = 0.957
- Training/Test gap: 3%

## 📊 Features Used
- Job Title, Industry, Location
- Experience Years, Education Level
- Skills Count, Certifications
- Company Size, Remote Work

## 🛠️ Tech Stack
Python · scikit-learn · Streamlit · pandas · joblib

## 📁 Project Structure
- app.py → Streamlit web app
- salary_predict.ipynb → Model training notebook
- Salary_prediction.pkl → Trained model
- columns.pkl → Feature columns
